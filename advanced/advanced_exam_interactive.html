1. What does super() do in Python inheritance?

class Child(Parent): def __init__(self, name): super().__init__(name) self.child_attr = "value"
✅ Correct Answer: B - Calls the parent class method
super() allows you to extend parent functionality rather than completely overriding it.

2. Which magic method is called when using the + operator?

result = obj1 + obj2
✅ Correct Answer: B - __add__()
__add__() enables operator overloading, allowing custom objects to work with the + operator.

3. What keyword is used to create a generator function?

def my_generator(): ??? 1 ??? 2 ??? 3
✅ Correct Answer: B - yield
yield creates a generator function that pauses execution and can be resumed, making it memory-efficient.

4. What does the regex pattern \d+ match?

✅ Correct Answer: B - One or more digits
\d matches any digit (0-9), and + means "one or more". So \d+ matches consecutive digits.

5. What keyword is used to define an asynchronous function?

✅ Correct Answer: A - async
"async" keyword defines an asynchronous function (coroutine). "await" is used inside async functions.
